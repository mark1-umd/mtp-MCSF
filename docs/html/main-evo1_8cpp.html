Detailed Description

A main program to drive the MCSF system to verify its trajectory generation capabilities.

Copyright
(c) 2017 Mark R. Jenkins. All rights reserved.
Author
MJenkins, ENPM 808X Spring 2017
Date
Mar 10, 2017 - Creation

This main program uses the Motion Control System Framework to generate a motion profile trajectory, then outputs the trajectory as a comma-separated value file in order to permit close comparison between the output of the trajectory generator in the MCSF and the model trajectory generator used to create the trajectory generation function in the MCSF.
